# Welcome to marketplace forecasting technical documentation!

This documentation is the repository of explanations for methodologies used in the marketplace forecasting project. The public of this documentation is the team of data scientists that work on the project. However, if you are a PL, PM, or any other role that is interested in the project, you are welcome to read it and ask questions.


## How to contribute to this documentation

First, check the [Retail Data Framework Documentation Service Guide](https://retaildocs.io/documentation-service/latest/guide/#/). It is important to understand how the documentation is built and how to contribute to it.

- Set your local machine according to [this instructions](https://docs.google.com/document/d/1VB35N1_txqnNN5CuYbE5CJTeRSi9fgXg8DGfw38iGsN/edit). It is important to install Retail Data Framework CLI to be able to create new doc versions!
- Clone `retail_demand-forecast-docs` repository
- Create a new branch from `main` with the name of the feature you are working on
- Make your changes, commit and create a Pull Request to `main`
- Ask for review of one member, and merge after approval.
- Run `retail create-version A.B.C` where `A.B.C` is the version of your doc. You can see the [latest version here](https://web.retailcloud.io/demand-forecast-docs/versions)
